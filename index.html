# Jogo da Velha Infinito Um jogo da velha com mecânica infinita onde, quando o tabuleiro fica cheio, as peças mais antigas de cada jogador são removidas automaticamente. ## Como Jogar 1. Clique em "Começar/Reiniciar" para iniciar o jogo 2. Os jogadores alternam entre X e O 3. Quando todas as casas estiverem preenchidas, o jogo automaticamente remove as peças mais antigas de cada jogador 4. Use "Pausar" para pausar o jogo a qualquer momento 5. O primeiro jogador a formar uma linha de 3 peças vence! ## Funcionalidades - Interface moderna com tema escuro - Linha de vitória animada - Sistema de remoção automática de peças antigas - Histórico de jogadas - Controles de pausa e reinício ## Tecnologias - HTML5 - CSS3 - JavaScript Vanilla
